About this project

Understanding a new codebase is often the slowest part of onboarding onto a project. This tool builds a code-aware retrieval pipeline over any public repository — extracting files, embedding them locally, and using an LLM to answer questions about structure, logic, and implementation — so developers can explore and understand large codebases faster.
